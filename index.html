org-mode

customize source block style

create a css for src: 

 pre.src {
    font-family: sans-serif;
    font-size: 95%;
    color: #93a1a1;
    background-color: #002b36;
    padding: 2px;
    border: #a7a7a7 1px solid;
    line-height: 120%
}
 div.src { 
    border: #a7a7a7 1px solid;
    background-color: #002b36;
    padding: 2px;
    overflow: auto;
}
 div.src {
    font-family: sans-serif;
    font-size: 95%;
    color: #93a1a1;
    padding: 0px;
    line-height: 120%;
    margin: 0px;
}
div.src p {
    font-family: sans-serif;
    font-size: 95%;
    color: #93a1a1;
    padding: 0px;
    line-height: 120%;
    margin: 0px;
}

add HTML head to org file 

#+HTML_HEAD: <link rel="stylesheet" type="text/css" href="src.css" />

Auto Hot Key

nested if statement

nested if is not allowed in ahk script: 

if a
{
   if b
   {
       ...
   }
}

you can wrap it to another function: 

fb(b)
{
   if b
   {
       ...
   }
}


if a
{
   fb(b)
}
