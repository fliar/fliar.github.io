* Windows
** Tools
*** Auto Hot Key
**** nested if statement
	 nested if is not allowed in ahk script:	 
	 #+BEGIN_SRC bash
	 if a
	 {
	 	if b
		{
			...
		}
	 }
	 #+END_SRC
	 you can wrap it to another function:
	 #+BEGIN_SRC bash
	 fb(b)
	 {
	 	if b
		{
			...
		}
	 }
	 

	 if a
	 {
	 	fb(b)
	 }
	 #+END_SRC
